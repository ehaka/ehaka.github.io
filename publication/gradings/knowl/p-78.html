Next, we claim that the map \(f\) is injective, which would prove that \(V_\alpha = W_{f(\alpha)}\) for all \(\alpha\in \Omega\) and so \(\mathcal V \) and \(\mathcal W\) would have the same layers. Note that since \(A \cong \mathbb{Z}^m\text{,}\) then for every nonzero \(\alpha \in A\) there exists a homomorphism \(\varphi \in A^*\) such that \(\varphi(\alpha)\neq 0\text{.}\) Therefore, if \(\alpha, \alpha' \in \Omega\) are such that \(f(\alpha) = f(\alpha')\text{,}\) then by the construction of the map \(f\) we have

\begin{equation*} \varphi(\alpha-\alpha') = \varphi(\alpha)-\varphi(\alpha') = f(\alpha)(\delta_\varphi)-f(\alpha')(\delta_\varphi) = 0 \end{equation*}

for every homomorphism \(\varphi\colon A\to F\text{.}\) So \(\alpha = \alpha'\) and \(f\) is injective.
