A problem of this form can be converted into a linear optimization problem as follows:

  1. replace \(\min_{\theta\in E_0^h} w(\theta)_\mathbf{a}\) with the new variable \(x\text{,}\) and add the constraint \(x\le w(\theta)_\mathbf{a}\) for each \(\theta\in E_0^h\text{;}\)
  2. replace \(\max_{\tilde{\theta}\in E_0^{h-1}}w(\tilde{\theta})\) with the new variable \(y\text{,}\) and add the constraint \(y\ge w(\tilde{\theta})_\mathbf{a}\) for each \(\tilde{\theta}\in E_0^{h-1}\text{;}\)
  3. normalise the expression by imposing \(Q_\mathbf{a}=1.\)

We are then left with the following expression for our original maximisation problem

\begin{gather*} \max_{\mathbf{a}\in\mathbb{R}^k_+,x,y}\big\{x-y\mid x\le w(\theta)_\mathbf{a},y\ge w(\tilde{\theta})_\mathbf{a},\forall\,\theta\in E_0^h,\tilde{\theta}\in E_0^{h-1},Q_\mathbf{a}=1\big\} \end{gather*}

which can be easily solved by a computer.
