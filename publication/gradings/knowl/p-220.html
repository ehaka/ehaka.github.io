Input: A grading \(\mathcal{V}\) for a Lie algebra \(\mathfrak{g}\) over any field. Output: Whether \(\mathcal{V}\) admits a positive realization or not.

  1. Form the \(N \times k\) matrix \(A\) and the vector \(\mathbf{b}\) associated with the problem.
  2. For each invertible \(N\times N\) matrix \(B\) formed from the columns of the matrix \(A\) do the following: Compute \(\mathbf{x}=B^{-1}(\mathbf{b})\text{.}\) If \(\mathbf{x} \ge 0\text{,}\) then return that the grading admits a positive realization.
  3. Otherwise, return that the grading has no positive realization.
